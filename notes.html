My HTML Notes

Block Elements

Block elements are used for large sections of text and create an invisible block around the text along with other default displays.

  • < p > - Paragraphs are separated from other texts and have a small margin.
  • < ul > & < ol > - Unordered & Ordered Lists
    • Unordered lists have bullet points and ordered lists have numbers.
  • < li > - List items inside the UL or OL list
  • < h1 > -- < h6 > - Section headers beginning from largest to smallest.
    • Used for headlines and section titles.
  • < div > - Divides the page and has no preset display settings so they're frequently used with CSS styling.

Inline Elements

Only modify the display of the text they surround.

  • < em > - Italicized
  • < strong > - Bold
  • < br > - Line break
  • < mark > - Highlight
  • < sup > - Super script
  • < sub > - Sub script
